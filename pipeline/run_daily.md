# إجراء التحديث — مرصد الشرق الأوسط (نشر مباشر، مرّتان يوميًا)

الروتين المجدول (وكيل Claude) يتبع هذا الإجراء **صباحًا ومساءً**. الأحداث الجديدة **تُنشر مباشرةً**
على الموقع العام بلا مراجعة.

## تقسيم العمل الآلي

| الخطوة | المنفّذ | التوقيت (الرياض) |
|---|---|---|
| ① الجلب | GitHub Actions — [`fetch_news.yml`](../.github/workflows/fetch_news.yml) | 7:30ص و7:30م |
| ②–⑤ التصنيف والدمج والدفع | روتين وكيل Claude المجدول (جلسة سحابية جديدة) | 8:00ص و8:00م |

سبب التقسيم: مواقع الأخبار محجوبة في بيئة الوكيل السحابية، بينما مشغّلات GitHub مفتوحة الإنترنت.
سير العمل يودع `pipeline/raw_items.json` + `pipeline/fetch_meta.json` (وقت الجلب وعدد الأخبار) في
`main`، والوكيل يسحبهما ثم يكمل. **قاعدة الحداثة:** إن كان `fetched_at` أقدم من 12 ساعة أو
`count=0` يتوقف الوكيل دون نشر.

## الخطوات

**① الجلب** *(آلي عبر GitHub Actions — لا تشغّله يدويًا إلا محليًا)*
```bash
python pipeline/fetch_sources.py
```
يكتب `pipeline/raw_items.json` (أخبار آخر 48 ساعة من RSS + GDELT، مُصفّاة بالصلة).

**② التصنيف (خطوة الوكيل)**
اقرأ `pipeline/raw_items.json`. لكل خبر حقيقي وذي صلة، أنشئ عنصرًا في `pipeline/classified.json`:
```json
{"التاريخ":"YYYY-MM-DD","الدولة":"إيران","المدينة":"طهران",
 "طبيعة":"عسكري / نووي","نطاق":"دولي","خطر":"حرج",
 "الحدث":"جملة عربية واحدة","التفاصيل":"جملة أو \"\"","المصدر":"اسم القناة"}
```
القواعد: طبيعة من {عسكري، سياسي، دبلوماسي، أمني، اقتصادي، إنساني، قانوني، تقني، اجتماعي، بيئي} (نووي→عسكري) · نطاق: إقليمي/دولي/داخلي · خطر: حرج/مرتفع جداً/مرتفع/متوسط/منخفض · لا تختلق حدثًا · تجاهل غير الجيوسياسي والمكرّر.

**③④ الدمج والنشر المباشر**
```bash
python pipeline/merge_rebuild.py
```
يضيف الأحداث الجديدة مباشرةً إلى `public/index.html` (بلا تكرار)، ويحدّث العدّادات. يطبع «نُشر: N».

**⑤ الدفع (ينشر تلقائيًا)**
```bash
git add -A && git commit -m "تحديث تلقائي: {التاريخ والفترة}" && git push
```
Netlify ينشر تلقائيًا من مجلد `public/` خلال دقيقة.

## البنية
- `public/index.html` — الموقع العام · مجلد النشر في Netlify.
- `pipeline/` — السكربتات (خارج النشر — لا يصلها الجمهور).
- الملفان `raw_items.json` و`classified.json` مؤقتان (في `.gitignore`).
