# مرصد الشرق الأوسط للتحليل الجيوسياسي

موقع ثابت مكتفٍ ذاتيًا (`index.html`) يرصد الأحداث الجيوسياسية الإقليمية على خريطة تفاعلية ومستكشف بيانات. كل البيانات مضمّنة داخل الملف في `<script id="EV">`.

**الرابط الحيّ:** https://famous-biscochitos-e29381.netlify.app

## التحديث اليومي الآلي

مجلد [`pipeline/`](pipeline/) يحوي خط أنابيب يجمع أخبار الشرق الأوسط يوميًا من RSS (الجزيرة، RT عربي، BBC عربي) وGDELT وبحث الويب، يصنّفها، يدمجها بلا تكرار، ويعيد بناء `index.html`. التفاصيل في [`pipeline/run_daily.md`](pipeline/run_daily.md).

**التشغيل مؤتمت بالكامل مرّتين يوميًا (7:00 صباحًا ومساءً بتوقيت الرياض):** روتين وكيل Claude مجدول يجلب ويصنّف ويدمج ويدفع إلى `main` فينشر Netlify تلقائيًا، ويعقبه (7:30) سير عمل [`fetch_news.yml`](.github/workflows/fetch_news.yml) الذي يودع حصاد الأخبار الخام في المستودع كطبقة احتياط وسجل.

| ملف | الدور |
|---|---|
| `pipeline/config.py` | المصادر، التصنيف الثابت، جدول الإحداثيات |
| `pipeline/fetch_sources.py` | ① جلب أخبار آخر 48 ساعة → `raw_items.json` |
| `pipeline/classify.py` | ② مساعد الترميز الحتمي (إحداثيات، أقنعة) |
| `pipeline/merge_rebuild.py` | ③④ دمج بلا تكرار + حقن في `index.html` |
| `pipeline/run_daily.md` | إجراء الروتين المجدول خطوةً بخطوة |
| `pipeline/admin.py` | 🛡️ لوحة الأدمن المحلية: بحث/تعديل/حذف أي حدث منشور ثم «حفظ ونشر» |
| `admin.bat` | تشغيل لوحة الأدمن بنقرة مزدوجة (تفتح على http://127.0.0.1:8765) |

## النشر

مربوط بـ Netlify عبر Git: **كل `git push` إلى الفرع الرئيسي يُطلق نشرًا تلقائيًا** على نفس الرابط.
